## 图片替换方案

### 方案一：手动替换（推荐）

1. **准备图片**
   - 格式：PNG 或 JPG
   - 角色尺寸：建议 400x500px
   - 武器尺寸：建议 600x200px 或 400x400px
   - 载具尺寸：建议 600x400px

2. **替换步骤**
   ```bash
   # 替换单个图片
   cp your-image.png /path/to/yh-wiki/public/images/characters/nanally.png

   # 批量替换后，运行转换脚本
   node scripts/convert-to-webp.js
   ```

3. **WebP 转换脚本**

   创建 `scripts/convert-to-webp.js`:
   ```javascript
   const sharp = require("sharp");
   const fs = require("fs");
   const path = require("path");

   const ROOT = path.resolve(__dirname, "..");
   const IMAGES_DIR = path.join(ROOT, "public/images");

   async function convertToWebp(dir) {
     const files = fs.readdirSync(dir);
     for (const file of files) {
       if (file.endsWith(".png") || file.endsWith(".jpg")) {
         const inputPath = path.join(dir, file);
         const outputPath = inputPath.replace(/\.(png|jpg)$/, ".webp");

         await sharp(inputPath)
           .webp({ quality: 85 })
           .toFile(outputPath);

         fs.unlinkSync(inputPath);
         console.log(`Converted: ${file} → ${path.basename(outputPath)}`);
       }
     }
   }

   ["characters", "weapons", "vehicles"].forEach(type => {
     convertToWebp(path.join(IMAGES_DIR, type));
   });
   ```

### 方案二：使用官方素材

关注游戏官方素材发布，获取官方授权的图片资源。

### 方案三：社区贡献

可以接受社区贡献者提交的图片，需确保：
1. 图片来源合法
2. 质量符合要求
3. 已转换为 WebP 格式

## 自动导入脚本

如果有可用的图片 URL，可以按以下步骤操作：

1. **编辑 image-sources.json**
   ```json
   {
     "_comment": "Add image URLs for each ID",
     "characters": {
       "nanally": "https://example.com/nanally.png"
     },
     "weapons": {
       "fangs-and-claws": "https://example.com/fangs.png"
     },
     "vehicles": {
       "drift-photon": "https://example.com/drift.png"
     }
   }
   ```

2. **运行导入脚本**
   ```bash
   node scripts/import-images.js
   ```
